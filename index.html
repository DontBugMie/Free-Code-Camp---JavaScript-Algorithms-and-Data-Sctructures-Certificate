Object Oriented Programming: Defining a Constructor Function

What is a Constructor?
  • A Function that creates a new object.
  • The Constructor defines behaviors and properties that will below to the new object.
  • This is the blueprint of objects, the adam to mankind.
  • A Constructor is defined using a capital letter - this is to distinguish between other functions
  • "this" is used in Constructors to set properties of the object they will create. Inside of the constructor "this refers to the new object it will create". "this" = refer to new object
  • Instances can be created from Constructor functions
